# DCA Proof-of-Action MVP Report

## 結果

DCA Proof-of-Action MVP 已完成。Baby Hippo 現在會把「建立計畫」與「真的開始執行」清楚分開，不會因為看過執行說明就自動判定完成定投。

整個功能維持本機模式：

- 沒有交易所 API
- 沒有 API Key
- 沒有錢包連接
- 沒有交易、簽名或自動執行
- 沒有後端
- 沒有修改 Aave Monitor

## 四階段行動進度

### Level 1 — Plan Created

- 使用者儲存第一份 DCA 計畫後獲得 `+5 BHP`
- 頁面明確說明：已建立計畫，但還沒有開始執行
- 下一步引導至 Binance 或 OKX 交易所選擇

### Level 2 — Exchange Selected

- 點擊 Binance 或 OKX 合作連結後獲得 `+5 BHP`
- 同一成就只會計分一次
- Ether.fi 不會被誤判為交易所選擇
- 完成後顯示下一步：在交易所建立現貨定期定額

### Level 3 — DCA Started

- 使用者必須開啟自我回報並確認五個項目：
  - 已選擇 BTC 或 ETH
  - 已設定固定金額
  - 了解這不是保證獲利
  - 沒有使用生活必需金
  - 沒有使用合約或槓桿
- 五項全部確認後才能送出
- 完成後獲得 `+20 BHP`
- 狀態顯示為「未驗證的自我回報」
- 明確警告：入金打合約不算 DCA

### Level 4 — On-Chain Growth Ready

- Level 3 完成後解鎖
- 不額外發放積分
- 說明交易所內資料無法被 Baby Hippo 看見
- 引導使用者未來把資產提領至自己的鏈上錢包後，再使用 Lobster Watch 觀察公開鏈上資料

## MVP 限制

- DCA Started 為未驗證的自我回報
- Baby Hippo 不會讀取交易所帳戶或購買紀錄
- 不會確認使用者是否持續執行每一期 DCA
- BHP 是本機教育成就積分，不是代幣，也沒有金融價值

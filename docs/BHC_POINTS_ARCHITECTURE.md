# BHC Points Architecture

## 目的

BHC Points 是 Baby Hippo 的「紀律紀錄系統」，不是代幣、加密貨幣、證券、投資商品或獲利承諾。

它只回答一個問題：

> 使用者是否正在建立對自己有幫助的長期習慣？

BHC Points 不獎勵賺多少錢、不比較資產規模，也不鼓勵高風險交易。普通工作者即使每次只投入小額，只要持續學習、記錄與管理風險，也應得到肯定。

## 核心原則

1. 獎勵紀律，不獎勵獲利。
2. 獎勵持續行動，不獎勵資金大小。
3. 教育與風險管理優先於投資行動。
4. 不因買入特定資產、使用推薦連結或增加交易量而給分。
5. 不把積分描述成未來空投、可兌現資產或保證獎勵。
6. 所有紀錄都應清楚標示是自行回報、系統紀錄或未來驗證。

## 建議得分規則

| 行動 | 分數 | 判定方式 |
|---|---:|---|
| 完成一次 DCA 紀律紀錄 | +10 | 使用者主動完成紀錄 |
| 完成一週 DCA 計畫 | +50 | 該週設定的提醒皆完成 |
| 連續完成四週 | +100 | 四個連續週期皆完成 |
| 完成一篇資產學習指南 | +20 | 閱讀後確認完成 |
| 完成風險測驗 | +20 | 完成測驗，不以高風險答案得分 |
| 建立緊急預備金目標 | +20 | 完成規劃，不要求實際存款證明 |
| 寫下本週學習或錯誤紀錄 | +10 | 完成反思紀錄 |

以下行為不得給分：

- 賺取投資報酬
- 增加投入金額
- 開槓桿或借款
- 頻繁交易
- 點擊交易所或推薦連結
- 購買指定代幣
- 邀請他人投入資金

## 等級系統

| 等級 | 建議門檻 | 意義 |
|---|---:|---|
| Explorer | 0–99 | 開始學習並建立第一個習慣 |
| Builder | 100–299 | 能持續執行基本計畫 |
| Operator | 300–599 | 能記錄、檢查並調整風險 |
| Founder | 600–999 | 為自己的長期計畫負責 |
| Chain Boss | 1000+ | 長期維持紀律，也願意幫助社群學習 |

等級代表學習與紀律里程碑，不代表投資能力、財富、信用或專業資格。

## 使用者看到的內容

積分頁應清楚顯示：

- BHC Points 總分
- 目前等級
- 下一個等級所需的紀律行動
- 學習、DCA、風險管理與反思的分類進度
- 每一筆得分原因
- 自行回報與系統紀錄的差異

固定聲明：

> BHC Points 不是代幣、加密貨幣、證券、投資商品或財務回報。積分沒有價格，也不保證能兌換任何獎勵。

## 驗證架構

### 第一階段：本機紀律紀錄

- 使用 localStorage
- 使用者自行標記完成
- 不連接錢包、交易所或銀行
- 清楚顯示「自行回報」

### 第二階段：帳號同步

- 經使用者同意後同步跨裝置進度
- 只儲存必要的任務狀態
- 不收集交易所密碼、助記詞或私鑰

### 第三階段：可選驗證

- 只在使用者主動同意下驗證習慣
- 優先驗證「是否完成固定行動」，不讀取完整財富資料
- 任何 API 或鏈上驗證都必須是唯讀
- 驗證失敗不得懲罰使用者

## 與產品的連接

### Lobster Watch

顯示 DCA 提醒完成率、連續週數與紀律里程碑。Lobster Watch 是紀律儀表板，不是淨資產競賽。

### DCA Reminder

建立提醒不代表完成。使用者在提醒後主動確認與記錄，才獲得 DCA 紀律分數。

### 學習專區

BTC、ETH、SOL、LINK、BNB、HYP、TAO、IO、DOGE 指南皆可成為學習任務。完成後應提醒使用者：學習不等於必須買入。

### 風險教育

風險測驗應獎勵理解以下原則：

- 不使用生活必需金
- 不借錢追高
- 不分享助記詞或私鑰
- 不相信保證獲利
- 不把歷史報酬當成未來承諾

## 防止錯誤激勵

- 不設每日無限刷分任務。
- 同一篇指南只給一次主要完成分。
- 連續紀律中斷後可重新開始，不使用羞辱或恐嚇文案。
- 排行榜若未來推出，預設比較完成率與互助貢獻，不比較投入金額。
- 不用倒數計時、FOMO 或「錯過就虧」鼓勵行動。

## 實作順序

1. 統一名稱為 BHC Points。
2. 建立新的任務與分數對照表。
3. 將 DCA 完成、每週完成、四週連續納入紀律紀錄。
4. 將九篇資產指南與風險測驗納入學習紀錄。
5. 建立清楚的積分明細與自行回報標籤。
6. 完成隱私與非金融產品聲明。
7. 經使用者測試後，再評估是否需要帳號同步。

## 功能審查問題

每個 BHC Points 新任務上線前都必須回答：

> 這個任務是否幫助努力生活的人建立長期紀律？

如果答案是否，就不應加入。

# Hippo Academy Content

Plain-language lessons, knowledge checks, and learning paths for the Baby Hippo community.

Initial subjects:

- Wallet and seed phrase safety
- Token and network basics
- DCA fundamentals
- DeFi lending and liquidation
- APY and long-term compounding
- Scam and pump-and-dump warning signs
- Understanding Lobster Watch alerts

Content should be short, respectful, beginner-friendly, and useful to people learning during limited free time. Learning rewards recognize participation and understanding, not transaction volume or speculation.

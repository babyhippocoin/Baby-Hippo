# Lobster Watch Prototype

A clickable, frontend-only Next.js prototype for the first Baby Hippo product.

## Run locally

Use Node.js 20 or newer and pnpm 11.5.3. This project uses pnpm as its only supported package manager.

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Production build

```bash
pnpm run build
pnpm start
```

All values are mock data. There is no backend, database, wallet connection, blockchain integration, or transaction capability.
